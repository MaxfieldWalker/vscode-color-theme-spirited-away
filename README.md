## Spirited Away Color Theme

![Screenshot](http://i.imgur.com/FZLCcYH.png)


## Install
Launch Visual Studio Code Quick Open (Ctrl+P), paste the following command, and press enter.

```
ext install vscode-color-theme-spirited-away
```
